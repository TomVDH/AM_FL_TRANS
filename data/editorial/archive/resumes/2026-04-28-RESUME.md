## 🔧 Known gotchas (engineering notes)

### Python 3.9 f-string limitation
F-strings can't contain backslashes in expressions. When using `re.findall(r'\bge\b', ...)` inside f-string, **pre-assign the regex result to a variable**:

```python
# BAD: print(f"ge: {len(re.findall(r'\\bge\\b', text))}")
# GOOD:
ge = len(re.findall(r'\bge\b', text))
print(f"ge: {ge}")
```

### Lookbehind requires fixed width
Python's `re` module rejects variable-width lookbehinds. For preposition-aware patterns, use **capture-group + backreference** instead:

```python
# BAD: re.sub(r'(?<=\b(?:voor|aan|met)\s)u\b', 'jou', s)
# GOOD: re.sub(r'\b(voor|aan|met|...)\s+u\b', r'\1 jou', s)
```

### V2 inversion -t drop for je-paradigm
When je is subject in V2 inversion, the verb drops the -t suffix:
- `denkt je` → `denk je`
- `kunt je` → `kun je`
- `bent je` → `ben je` (also need to re-check after zijt→bent flip)

**Exclusion list (don't drop -t):** `dat, wat, het, niet, met, tot, ook, maar, moet, heet, eet, zit, weet, zat, echt, recht, kort, want, omdat, terwijl`. These are non-verbs OR stem-final-t verbs. Add `omdat` if missing — bug from Sturdy retcon.

### V2 inversion -t add for ge-paradigm (reverse direction)
When flipping je→ge for Kick-style chars, V2 inversion needs -t added:
- `denk je` → `denkt ge`
- `kan je` → `kunt ge`
- `ben je` → `zijt ge` (Flemish-specific verb form)
- `mag je` → `moogt ge`
- `hoor je` → `hoort ge`

Use a verb-mapping table; not all verbs need -t added (stem-final-t verbs like `eet`, `weet`, `zit` keep their form unchanged).

### Uppercase variants
Mechanical regex needs to handle ALL of: lowercase (`je`), title case (`Je`), all-caps (`JE`). The post-apply scan WILL catch residuals — fix them as a follow-up corrections file before declaring done.

### Engine-tag preservation
xlsx cells may contain engine tags like `{vpunch=100,0.75}`, `{shake}` etc. at line start. **Preserve verbatim.** Snapshot generation may strip them — verify against live xlsx before correcting.

### Capitalization rule for sentence-start contractions
Sentence-start `'t` or `'k` → next word IS capitalized. Examples: `'t Was niet op slot.` (not `'t was`). `'k Heb een nieuwe strategie.` (not `'k heb`). When flipping `'k → Ik`, normalize the next word to lowercase.
