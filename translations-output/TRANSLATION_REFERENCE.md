## Translation Style Notes

1. **Wordplay**: Many character names use puns on "ass" (donkey/buttocks). The Dutch translations cleverly use "Ezel" (donkey), "Zak" (sack/person), "Gast" (guy), and other creative terms.

2. **Tone**: The game has an irreverent, satirical tone. Translations should maintain this playful, sometimes crude humor.

3. **Alliteration**: Where possible, maintain alliterative patterns (e.g., "Trouwe" and "Triestige" both start with "Tr").

4. **Cultural Adaptation**: Some names are fully adapted rather than literally translated (e.g., "Melvin" → "Zeno", "Marcos" → "Tom").

5. **Formality**: Generally informal/casual tone throughout.

6. **Variables**: Preserve game variables like `{$NewName}` exactly as they appear.

7. **Line Breaks**: Preserve `\n` for line breaks in UI text.

## Common Patterns

### Political/Labor Terms
Based on the themes in E0 questions, expect terms related to:
- Manual labor (handarbeid)
- Working conditions (arbeidsomstandigheden)
- Strikes (stakingen)
- Redundancy (overtolligheid)

### Question Response Patterns
- Yes → Ja
- No → Nee
- I don't know → Ik weet het niet

### Time/Duration
- PAUSE → PAUZE
- duration → duur
- whenever → wanneer dan ook
